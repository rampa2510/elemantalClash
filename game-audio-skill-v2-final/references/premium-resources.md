## Conversion Workflow (Raw → Game-Ready)

### Step 1: Organize Downloads
```
raw-audio/
├── sonniss-gdc/
├── helton-yan/
├── kenney/
├── opengameart/
└── other/
```

### Step 2: Select Best Sounds
Listen and pick the best sounds for each category needed.

### Step 3: Normalize Loudness
```bash
ffmpeg -i input.wav -af loudnorm=I=-16:TP=-1.5:LRA=11 normalized.wav
```

### Step 4: Convert to Web Formats
```bash
# SFX (mono, small)
ffmpeg -i input.wav -ac 1 -c:a libopus -b:a 64k output.webm
ffmpeg -i input.wav -ac 1 -c:a libmp3lame -b:a 96k output.mp3

# Music (stereo, higher quality)
ffmpeg -i input.wav -c:a libopus -b:a 128k output.webm
ffmpeg -i input.wav -c:a libmp3lame -b:a 128k output.mp3
```

### Step 5: Create Audio Sprites
```bash
npm install -g audiosprite
audiosprite -f howler2 -e "webm,mp3" -o sprite-name *.wav
```

### Step 6: Implement with Howler.js
See `browser-implementation-guide.md` for complete code.

## Quick Command Reference

```bash
# Download all Sonniss GDC bundles (if using wget)
# Note: Check sonniss.com for current download links

# Batch convert folder to WebM
for f in *.wav; do
  ffmpeg -i "$f" -ac 1 -c:a libopus -b:a 64k "${f%.wav}.webm"
done

# Batch normalize loudness
for f in *.wav; do
  ffmpeg -i "$f" -af loudnorm=I=-16:TP=-1.5:LRA=11 "normalized_${f}"
done

# Create sprite from folder
audiosprite -f howler2 -e "webm,mp3" -b 64 -o output-sprite *.wav
```
